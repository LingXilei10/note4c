## 关键字

系统预定义的语义符

## 标识符

字母+_开始  字母、下划线、数字《32位

## 整型

### int

4字节

前缀+0：八进制

+0x：16进制

### bool

true和false

### enum

标识符表符号表示的数据

## 浮点型

指数示数法：整型/浮点型 + E +整型

### float

4位 需要标识f/F

### double

8位 默认小数是这个类型

## 字符型

1字节 -128~127

unsigned char 0-255    U后缀

'A'65    'a'97

"x"和'x'不同 (相差'\0')

\\.是转义字符，如：

\o 八进制   \ 



## 修饰符

short：int  2字节 

long：int和double  L后缀

signed和unsigned：char int 

## 优先级

|      | 运算符           | 功能                                                     |
| ---- | ---------------- | -------------------------------------------------------- |
| 1    | ::   []   .   -> | 成员选择                                                 |
| 2    | ++ --            | 自增自减                                                 |
|      | & *              | 取地址取值                                               |
|      | ! ~              | l逻辑非 按位反                                           |
|      | + -              | 取正取负                                                 |
|      | sizeof           | 运算符(不是函数)编译时计算 矩阵有、指针无 返回无符号整数 |
|      | new delete       |                                                          |
| 算数 | * / %            | %计算两个                                                |
|      | + -              |                                                          |
|      | << >>            | 左移位 右移位  l+()                                      |
| 关系 | < > == !=        |                                                          |
| 逻辑 | & ^ \|           | 按位                                                     |
|      | && \|\|          | 先处理左边 \|\|若左边符合条件，右边不执行                |
| 条件 | ?:               | 三元条件运算符                                           |
| 赋值 | = +=             | x=y=3 从右往左                                           |
| 逗号 | ,                | 值为最右                                                 |
